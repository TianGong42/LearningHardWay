# 第十六章
- close:关闭文件。跟你的编辑器中的"文件"->"保存"是一个意思
- read:读取文件内容。你可以把结果赋值给一个变量
- readline： 只读取文本文件中的一行
- truncate: 清空文件，小心使用
- write('stuff'):将"stuff"写入文件
- seek(0):将文件位置移动到文件开头
